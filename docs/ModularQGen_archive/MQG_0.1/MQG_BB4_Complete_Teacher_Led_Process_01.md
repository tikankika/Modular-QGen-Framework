# Building Block 4: Complete Documentation
## Three-Stage Model with Teacher-Led Dialogic Process

---

# EXECUTIVE SUMMARY: TEACHER LEADS, AI SUPPORTS

Building Block 4 is built on a fundamental principle: **The teacher makes all pedagogical decisions through structured dialogue with the AI system**. AI proposes, analyzes, and implements - but the teacher decides.

---

# PART 1: CORE PROCESS ARCHITECTURE

## The Three-Stage Model: Overview

```
STAGE 4A: BASIC GENERATION
├─ Focus: Content creation
├─ Dialogue: Explorative, creative
├─ Teacher's role: Directs content focus
└─ AI's role: Proposes and generates

STAGE 4B: DISTRIBUTION REVIEW  
├─ Focus: Systematic analysis
├─ Dialogue: Analytical, decision-driven
├─ Teacher's role: Makes strategic decisions
└─ AI's role: Presents data and analysis

STAGE 4C: FINALIZATION
├─ Focus: Technical implementation
├─ Dialogue: Practical, solution-oriented  
├─ Teacher's role: Approves format and quality
└─ AI's role: Implements efficiently
```

## The Dialogic Process

### Why Dialogue + Structure = Quality

**Traditional method:** Chaotic process where everything mixes
**Three-stage model:** Structured conversations with clear purposes

### Teacher's Decision Authority Throughout

**In Stage 4A:** Teacher decides:
- What content to assess
- Which examples to use
- Difficulty level and depth
- Pedagogical direction

**In Stage 4B:** Teacher decides:
- Whether distribution is acceptable
- What adjustments are needed
- Priorities between goals
- When to proceed

**In Stage 4C:** Teacher decides:
- Which question format fits
- Quality of distractors
- Feedback design
- Final approval

# PART 3: WHY THREE STAGES WITH TEACHER CONTROL

## The Path to Effective Question Production

### From Chaos to Clarity
The three-stage model gives you as a teacher a clear structure where you can focus on one thing at a time. This frees your cognitive capacity for what really matters - pedagogical quality.

### Your Guide Through the Process

**Stage 4A: Create Content Freely**
```
✓ Focus entirely on WHAT students should learn
✓ Let pedagogical ideas flow without technical constraints  
✓ AI helps you generate variations on your themes
✓ You direct the content's direction and depth
```
*Result: Rich content matching your vision*

**Stage 4B: Get Overview and Decide**
```
✓ See the complete picture of what you've created together
✓ Identify gaps or imbalances immediately
✓ Make informed decisions about adjustments
✓ Approve when distribution meets your goals
```
*Result: Balanced assessment without surprises*

**Stage 4C: Implement Efficiently**
```
✓ Format only approved content
✓ Choose question types that fit your purpose
✓ Create feedback that supports learning
✓ Finalize with technical precision
```
*Result: Professional questions ready for use*

### The Benefits for You as Teacher

**Cognitive Freedom**
- One focus at a time = deeper quality
- No split attention
- Space for pedagogical creativity

**Control and Overview**  
- You see the whole before detail work
- No late surprises
- All decisions are conscious and informed

**Time Efficiency**
- Right from the start instead of extensive rework
- AI's strengths used optimally
- Your time goes to pedagogical decisions, not administration

### How to Maximize the Process

1. **Trust the structure** - It exists to support you
2. **Use AI as partner** - Let it handle the systematic
3. **Maintain focus** - One stage, one purpose, full concentration
4. **Make decisions with confidence** - You have all information you need

### The Result: Quality Through Control

When you follow the three-stage model you get:
- Questions that truly test what you intend
- Balanced distribution without hassle
- Professional finish on first attempt
- More time for teaching, less for administration

**Your pedagogical expertise + AI's systematic power = Optimal assessment quality**

# PART 4: DIALOGUE EXAMPLES THROUGHOUT

## Stage 4A: Generative Dialogue (Teacher Leads)

```
Teacher: "We need questions on ecosystem services for 
         master's students in environmental engineering."

AI: "I can focus on:
     1. Theoretical concepts
     2. Practical applications
     3. Valuation methods
     What's your priority?"

Teacher: "Mainly 2 and 3, students know the theory. 
         Use Swedish case studies."

AI: "Generating questions on practical applications 
     with examples from Gothenburg and the West Coast..."

Teacher: "Good, but also include urban examples,
         not just marine."

AI: "Adjusting the ecosystem balance..."
```

## Stage 4B: Analytical Dialogue (Teacher Decides)

```
AI: "Analysis of 45 generated questions:
     - Learning objective 3 has only 1 question (minimum 2)
     - 15% at analysis level (target was 25%)
     - Good distribution of difficulties
     
     Recommend: Add 3-4 analysis questions"

Teacher: "Which topics lack analysis questions?"

AI: "Ecosystem valuation and trade-offs between
     services lack deeper analysis."

Teacher: "Add 2 on valuation, 2 on trade-offs.
         Remove 2 of the simplest factual questions."

AI: "Implementing your decisions..."
```

## Stage 4C: Implementation Dialogue (Teacher Approves)

```
AI: "Question 23 on 'economic value of ecosystem services'
     - could be essay or multiple choice. Your preference?"

Teacher: "Multiple choice, but with realistic figures from
         actual valuations."

AI: "Creating options based on:
     A: Cost of artificial purification (correct)
     B: Tourism revenue (too low)
     C: Property values (too high)
     D: Carbon storage (different service)"

Teacher: "Good distractors. Add explanation about
         why A is most relevant."

AI: "Implementing with pedagogical feedback..."
```

# PART 7: THE ITERATIVE DIALOGIC PROCESS

## Iteration Patterns with Teacher Control

### Pattern 1: Refinement Through Dialogue
```
AI proposes → Teacher guides → AI adjusts → Teacher approves
```

### Pattern 2: Problem-Solving Together
```
AI identifies issue → Teacher determines approach → 
AI implements → Teacher validates
```

### Pattern 3: Quality Enhancement
```
Teacher gives feedback → AI revises specifically → 
Teacher evaluates → Iteration if needed
```

## Teacher's Decision Points

**Critical decisions that ALWAYS belong to the teacher:**
1. Pedagogical direction and goals
2. Content priorities
3. Difficulty calibration
4. Approval between stages
5. Quality standards
6. Final approval

**AI must NEVER:**
- Make pedagogical decisions independently
- Change teacher's priorities
- Proceed without approval
- Compromise quality for efficiency
